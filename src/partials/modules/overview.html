TurboPutative is a tool that simplifies the matrices with the possible identifications obtained in untargeted metabolomics studies based on liquid chromatography coupled with mass spectrometry (with high-resolution analyzers) using electrospray ionization (LC/ESI-MS). In this type of studies, the matrices with the candidate metabolites are obtained using tools such as Ceu Mass Mediator 3.0 or MetaboSearch, which compare the monoisotopic mass of the features detected with the masses of the compounds contained in the databases. Despite using high-resolution analyzers, multiple candidate metabolites can be assigned to the same feature due to the similarity in the compounds masses, which in the case of isomers is identical. Even compounds with different mass, when ionized with different adducts, can generate ions with very similar m/z ratios, so they could be assigned to the same feature.

Therefore, after obtaining the matrices with the possible identifications, the researcher must determine which candidate metabolites are most likely, which will allow him to proceed with the interpretation of the results, or prioritize the purchase of authentic chemical standards or analyzes in MS/MS mode. Consequently, researchers must manually review each entry to compare their associated errors and adducts, and frequently search databases to identify possible dietary, drug, or microorganism compound. Since this process consumes a lot of time and effort, we have developed TurboPutative, which makes it possible to streamline the handling and simplification of the tables with the possible identifications to facilitate their visualization and the extraction of useful information, with the aim of identifying the most probable candidate metabolites. The code of the program and the desktop version are available on GitHub.
